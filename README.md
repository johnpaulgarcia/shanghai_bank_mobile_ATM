## 技术栈：
  react + redux + webpack + react-router + ES6/7/8 + immutable


## 运行项目（nodejs 6.0+）

```
 git clone https://github.com/bailicangdu/react-pxq.git

 cd react-pxq

 npm i  或者运行  yarn(推荐)
  
 npm start

 npm run build （发布）
```


## 说明

>  本项目主要用于理解 react 和 redux 的编译方式，以及 react + redux 之间的配合方式

>  开发环境 macOS 10.13.1  Chrome 63  nodejs 8.9.1
