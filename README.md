## 技术栈：
  react + redux + webpack + react-router + ES6/7/8 + immutable


## 运行项目（nodejs 6.0+）

```
 git clone https://github.com/bailicangdu/react-pxq.git

 cd react-pxq

 npm i  或者运行  yarn(推荐)
  
 npm start

 npm run build （发布）
```
